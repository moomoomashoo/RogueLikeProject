Introduction


For my final year project, I chose to create a Procedurally Generated Roguelike game. Inspired by games such as "Rogue Legacy" and "Spelunky", I made a sidescrolling action platformer written in Python


Using the Pygame library, I made a simple combat and movement system, and populated each randomised level with a selection of enemy types.


The game was designed to be played with a controller, however keyboard controls have been implemented.


This game is a work-in-progress, however it is currently fully playable. Next steps include adding sprite-based graphics and a simple upgrade system.
